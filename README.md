# Touchstone Conference

Website for the Touchstone Conference, a national Christian event featuring leading speakers and thought leaders.

Live: https://thetouchstoneconference.com

---

## Overview

Designed and developed a conference website to support event promotion, speaker visibility, and attendee registration.

## Features

- Speaker and schedule presentation  
- Event promotion and information architecture  
- Registration support  

## Tech Stack

- Laravel (PHP)  
- Bootstrap 5  
- JavaScript  

## Role

Full design and development, including layout, frontend implementation, and deployment.
